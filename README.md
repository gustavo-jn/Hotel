# Sistema de Hotel

Sistema web desenvolvido para gerenciamento básico de um hotel, incluindo cadastro de usuários, autenticação e organização de páginas do sistema.

---

## Sobre o projeto

Este projeto simula um sistema de gestão de hotel, com foco em funcionalidades essenciais de um sistema real, como:

- Cadastro de usuários  
- Login/autenticação  
- Organização de páginas  
- Estrutura modular de sistema web  

O objetivo foi praticar desenvolvimento web com foco em estrutura de aplicação, organização de código e integração com banco de dados.

---

##  Tecnologias utilizadas

- PHP  
- HTML  
- CSS  
- JavaScript  
- Banco de dados (SQL)  

---


## Funcionalidades

-  Sistema de login  
-  Cadastro de usuários  
-  Navegação entre páginas  
-  Organização modular do código
